Equations and what they mean
Notation (symbols)
D+ = actually positive
D- = actually negative
T+ = predicted (tested) positive
T- = predicted (tested) negative
Base quantities
P = N * π
Actual positives in the dataset.
Nneg = N * (1 - π)
Actual negatives in the dataset.
TP = P * TPR
True positives (positive cases found correctly).
FN = P - TP
False negatives (positive cases missed).
FP = Nneg * FPR
False positives (negative cases predicted as positive).
TN = Nneg - FP
True negatives (negative cases correctly rejected).
Rates and scores
TPR (Recall) = TP / (TP + FN)
Among actual positives, how many you catch.
FPR = FP / (FP + TN)
Among actual negatives, how many false alarms you raise.
Precision (PPV) = TP / (TP + FP)
Among predicted positives, how many are truly positive.
Accuracy = (TP + TN) / N
Overall fraction of correct predictions.
F1 = 2 * (Precision(PPV) * Recall) / (Precision(PPV) + Recall)
Harmonic mean of Precision (PPV) and Recall.
Specificity (TNR) = TN / (TN + FP) = 1 - FPR
Among actual negatives, how many you correctly reject.
Balanced Accuracy = (TPR + TNR) / 2
Average of sensitivity and specificity.
NPV (Negative Predictive Value) = TN / (TN + FN)
Among predicted negatives, how many are truly negative.
Bayesian view: why prevalence matters
D+ = true positive class
D- = true negative class
T+ = model/test predicts positive
T- = model/test predicts negative
P(A|B): probability of A, assuming B happened
P(D+|T+): if test (model, prediction) is positive, chance it is truly positive class
Bayes links test quality (TPR/FPR) with prevalence (prior probability).
Prior: P(D+) = π, P(D-) = 1 - π
Before seeing a test result, prevalence is your best guess of disease/class probability.
Posterior after a positive test
P(T+|D+) = TPR
P(T+|D-) = FPR
P(D+|T+) = (TPR * π) / ((TPR * π) + (FPR * (1 - π))) = PPV
This is Precision (PPV): probability of truly positive given a positive prediction.
In words: start from prior positives (π), then adjust by how often positives/negatives trigger a positive test.
Posterior after a negative test
P(T-|D-) = TNR = 1 - FPR
P(T-|D+) = FNR = 1 - TPR
P(D-|T-) = (TNR * (1 - π)) / ((TNR * (1 - π)) + (FNR * π))
This is NPV (Negative Predictive Value): probability of truly negative given a negative prediction.
In words: among negative test results, compare true negatives against missed positives.
Practical takeaway
When π is very small, even a modest FPR can create many false positives, lowering PPV.
That is why Precision (PPV) can drop sharply in imbalanced settings, even when TPR looks strong.
